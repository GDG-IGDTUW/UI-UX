## Eklavya (एकलव्याय शिक्षां समर्पयामः) – Project Overview

Project Maintainer: Deepanshi Dhiman | Shriya Sandilya
Project Mentor: Sneha Pandey

## Description

Eklavya is a student-centric learning and support platform designed to help school
and college students manage academics, build strong peer networks, and access
personalized guidance within a single application.

Inspired by the spirit of self-learning and community support, Eklavya combines
study assistance, community discussions, e-library resources, and an AI-powered
mentor bot in a simple and intuitive interface.

## Examples for Contributors

To improve usability, accessibility, and overall design consistency,
contributors can work on the following areas:

Navigation & Layout Improvements

Improve navigation consistency across all screens

Simplify user flows for quicker access to key features

Component Standardization

Standardize card components such as notices, workshops, community posts, and
profile sections

Improve reusability and consistency of UI components

Community & AI Mentor Redesign

Redesign the Community screen to reduce visual clutter and clarify user
journeys

Improve the AI chat interface to feel more approachable and supportive

Progress Analytics Enhancement

Simplify progress visualizations for easier interpretation

Improve clarity of charts and performance indicators

Visual & Accessibility Enhancements

Improve color balance and text contrast

Refine typography hierarchy and spacing for better readability

## Requirements for Contributors

To maintain quality and consistency, contributors should focus on:

Usability:
Ensure the interface is intuitive, student-friendly, and easy to navigate across
devices.

Visual Hierarchy:
Maintain consistency in typography, spacing, alignment, and component usage for
a clean and professional appearance.

Accessibility:
Follow accessibility best practices including readable font sizes, sufficient
color contrast, and clear touch targets.

## Guidelines for Contributors

Align all designs with the existing Eklavya design language and component
system

Prioritize clarity, simplicity, and student comfort over visual complexity

Propose thoughtful design improvements that enhance focus and reduce cognitive
load

Test designs for responsiveness and consistency across different screen sizes
and devices


This project offers contributors an opportunity to work on a meaningful,
student-impact-driven platform, strengthen UI/UX and product thinking skills,
and contribute to an inclusive academic support ecosystem.
